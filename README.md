# Padel court keypoint detector

This is an implementation of a keypoint detector in padel courts. It includes the implementation of a ResNET-18 and a label feature to make the creation of annotation easier and faster.

## Installation

(Optional but recommended) Create a python conda environment and activate it:
```
conda create -n kpdetector
conda activate kpdetector
```

Clone the repository
```
git clone https://github.com/puccj/keypoint-detector
```
Install the requirements
```
pip install -r requirements.txt
```
